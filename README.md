# Blog
To Run the app (server)
Make sure you are in - Blog/ directory & type the following command
$ npm install
$ npm run app




Blog_Client
To Run the app (Client - React)
Make sure you are in - Blog_client/ directory & type the following command
$ npm install
$ npm start
